# 🌱 Kisaan Netra  

**Kisaan Netra** is an **AI + IoT powered smart agriculture system** that helps farmers optimize crop health through **soil condition detection** and **pest removal**.  
The system integrates **Machine Learning (ML)** models with **IoT-enabled sensors and actuators**, making farming more efficient, sustainable, and cost-effective.  

---

## 🚀 Features  
- 🔎 **Soil Detection**: Uses IoT sensors (moisture, pH, temperature) and ML models to analyze soil health.  
- 🐛 **Pest Detection & Removal**: Identifies pests using AI-powered image recognition and triggers IoT-based control mechanisms (like sprayers or ultrasonic repellers).  
- 📊 **Real-time Monitoring**: Live data from fields accessible through a mobile/web dashboard.  
- 🤖 **Smart Recommendations**: Suggests irrigation cycles, fertilizer use, and pest control strategies.  
- 🌍 **Scalable**: Works for small farms to large-scale agricultural fields.  

---

## 🛠️ Tech Stack  

### **Machine Learning (ML)**
- Python, TensorFlow / PyTorch  
- Image classification for pest detection  
- Soil classification & prediction models  

### **IoT Layer**
- Sensors: Soil moisture, pH, temperature, humidity  
- Actuators: Sprayers, water pumps, pest control devices  
- Microcontrollers: Arduino / ESP32 / Raspberry Pi  

### **Backend & Dashboard**
- Node.js / Flask / Django  
- MongoDB / Firebase for data storage  
- React / Next.js for frontend visualization  

---

## ⚙️ System Architecture  

1. **IoT Sensors** collect soil & environmental data.  
2. Data is sent to the **ML models** for soil health classification & pest detection.  
3. **Decision Engine** processes results and triggers **actuators** (e.g., pest removal, irrigation).  
4. Farmers can monitor and control operations via a **dashboard / mobile app**.
